# 🪐 Gravity Simulation

Interactive and visual physics simulation to explore the mechanics of gravity !

### 🚀 Features
* **Celestial Body Management:** Add planets and observe their orbits in real time.

* **Customizable Physics Engine:** Enable/disable collisions, manage bounces, and adjust the contact threshold via a dedicated interface.

* **Total Control:** Pause the simulation, advance frame by frame, or reset the space at any time.

* **Soundscape:** Integrated music player with volume control to complement the visual experience.
